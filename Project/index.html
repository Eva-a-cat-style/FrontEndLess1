Программирование на Java

Java — это мощный язык программирования, используемый для создания различных приложений, от веба до мобильных и десктопных программ. Приложения Java обычно транслируются в специальный байт-код, поэтому они могут работать на любой компьютерной архитектуре, для которой существует реализация виртуальной Java-машины. Дата официального выпуска — 23 мая 1995 года.

Преимущества Java

  • Многоплатформенность (Write Once, Run Anywhere)
  • Большое сообщество разработчиков
  • Богатая стандартная библиотека
  • Безопасность и производительность

Приложения, созданные на Java:

  • eBay
  • Amazon
  • PayPal
  • Twitter
  • Facebook
  • И даже официальный сайт Пентагона!
